Launch the requirement-analyzer subagent to analyze requirements.

## Usage
When user wants to understand the scope and scale of their request.

## Next Commands
After requirement analysis, consider:
- `/create-product-requirements` - Create product requirements document (Medium/Large features)
- `/create-technical-design` - Create technical design document (Medium/Large features)
- `/create-work-plan` - Create work plan (Any size)
- `/generate-tests` - Generate tests (if specs are ready)
- `/implement-task` - Start implementation (Small features)
